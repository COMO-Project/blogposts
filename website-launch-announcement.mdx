株式会社 COMO のコーポレートサイトが公開されました。
当社は COMO（コモ）と言います。
会社名を決定する際、全生物共通の細胞である LUCA「Last Universal Common Ancestor」（最終普遍共通祖先）という単細胞生物の名前を採用することを考えました。
しかし、同じ生物を日本の学者の論文では Comonote（コモノート：Common Note）と紹介されていたため、可愛らしい音である COMO という名称にしました。

生物学はロマンがいっぱいな学問であり、当社のブログでも（私の個人的な趣味で）生物学のトピックを紹介する予定です。

## 会社のミッション

「価値をつなぐ　仕組みを作る」を会社のミッションとして掲げています。
　デジタル技術の発展は、あらゆる価値を時間や場所をつなぎ、これまで制約となっていた数々の環境を飛び越えて新たな可能性を生み出すことになりました。個々の問題として一人で苦しみ悩んでいたようなことも、不可能だとして諦めていたことも、そういう世の中だとして受け入れていたことも、価値をつなぎ仕組みを作り出すことによって新たな解決策が生み出されるようになります。

当社は、自律的に組織化された仕組みを構築し、その仕組みを基盤に新たな価値を創造することを目指しています。
価値の組み合わせには無限の可能性があり、それによって築かれる仕組みも無限の可能性を秘めています。

私たちは、IT の技術を使って、社会問題を解決していくような自律的な仕組みづくりを行なっていきます。

## 会社のサービス

また、COMO では、WEB 制作やマーケティングの分野に限らず、デザイン・ブランディング・DX 推進において必要となる様々な分野に対応できるスキルを持ったメンバーたちがおり、お客様のニーズに応えた幅広いサービスを提供することができます。

さらに、中小企業の DX 推進支援のために、自社開発の WEB アプリケーション開発事業も展開しています。
このサービスでは、中小企業にとって必要不可欠な業務の効率化や新たなビジネスモデルの構築を支援するための WEB アプリケーションを開発しています。
お客様とともに課題を解決し、ビジネスの成長を支援することが COMO のミッションであり、自社開発の WEB アプリケーション開発事業もその一環として行っています。

COMO は、デジタル技術の発展によって生まれた新たな価値を創造し、社会課題の解決に貢献することを目指しています。お客様とともに課題を解決し、共に成長していくことを大切にし、スキルや知識を磨き続けることで、より良いサービスを提供できるよう努めています。

## このウェブサイトの特徴

- 言語：Next.js
- ブログ記事：markdown
- ３ D モデル：Blender
- アニメーション：react-three/fiber、react-three/drei、Framer Motion、GSAP

3D アニメーションは非常に面白い技術であり、今後もその活用方法を探求していく予定です。
